## **React Weather Application**




A Weather Application, where a user can:

Search for current weather by city.

Select an example location city to try out.

**Dependencies**
 
-     axios: "0.15.3",
-     express "4.14.1",
-     react "0.14.7",
-     react dom "0.14.7",
-     react-router "2.0.0"
 
 
 
**Get Started**

Install modules

> $ npm install


Run Server

> $ node server.js


Run webpack

> $ webpack -w


OR/AND

View live [here on Heroku](http://weatherapp-moshe.herokuapp.com/)
